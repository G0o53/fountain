# To Install
## Dependencies:

- [golang](https://go.dev/)
- [make](https://www.google.com/url?sa=t&source=web&rct=j&opi=89978449&url=https://www.gnu.org/software/make/&ved=2ahUKEwjqvpDatOyRAxVQulYBHfwUF6cQFnoECBcQAQ&usg=AOvVaw1zM-Js4YqRdqmHDbyhZJQE)

## Install - With 3 easy steps!

simply, clone with this command
```bash
git clone https://github.com/G0o53/fountain
```
then, simply enter the newly cloned directory with this command
```bash
cd fountain/src/fountain
```
then, lastly, just type (note, this may ask for your password)
```bash
make
```
and you should have fountain ready to go in /usr/local/bin/fountain and in /usr/local/bin/ff
